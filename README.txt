The official Aweza plugin for Wordpress.

== Description ==

The Aweza plugin for Wordpress enables the Aweza Popup of Awezified content automatically. Simply enter your Aweza API
credentials under Settings in your Wordpress admin and add aweza-data tags to content to begin.

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress
2. Enter your Aweza API credentials in Settings -> Aweza
